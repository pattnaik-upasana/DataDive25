# Data Directory

This directory contains the DuckDB database and any downloaded data files.

## Database Structure

The `digital_jobs.duckdb` database contains the following tables and views:

### Tables

- **`wb_indicators`**: World Bank indicator data including:
  - ICT services employment
  - ICT manufacturing employment
  - ICT goods exports/imports
  - Mobile subscriptions
  - Internet users
  - Broadband subscriptions

- **`digital_jobs`**: Sample digital/AI jobs data with:
  - Country code and name
  - Year
  - Industry
  - Skill type
  - Demand index
  - Supply index
  - Gap (demand - supply)

### Views

- **`country_trends`**: Aggregated country-level trends
- **`industry_trends`**: Aggregated industry-level trends
- **`skill_trends`**: Aggregated skill-level trends
- **`rising_lagging_countries`**: Analysis of rising vs. lagging countries

## Data Sources

Data is loaded from:
- World Bank API (via `load_data.py`)
- Sample generated data for demonstration

## Note

The database file (`digital_jobs.duckdb`) is generated by running `load_data.py` and is not included in version control (see `.gitignore`).
