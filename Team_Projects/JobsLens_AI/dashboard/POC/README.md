# Dashboards

This folder contains interactive HTML dashboards for visualizing AI job displacement analysis.

## Files

- **dashboard_index.html** - Main dashboard index page with navigation and key findings
- **dashboard_1_vulnerability_overview.html** - Vulnerability scores across countries
- **dashboard_2_sector_analysis.html** - Employment by sector analysis
- **dashboard_3_occupation_education.html** - Occupation risk and education analysis
- **dashboard_4_country_comparison.html** - Interactive country comparisons

## Viewing the Dashboards

Open [dashboard_index.html](dashboard_index.html) in your web browser to access all dashboards.

From the root of the repository:
```bash
open dashboards/dashboard_index.html
```

Or visit: [https://datacommunitydc.github.io/DataDive25/dashboards/dashboard_index.html](https://datacommunitydc.github.io/DataDive25/dashboards/dashboard_index.html)

## Features

All dashboards are fully interactive with:
- Hover tooltips for detailed data
- Zoom and pan capabilities
- Legend filtering
- Export to PNG functionality

## Data Source

Generated from World Bank I2D2 Database (2020-2021 surveys) covering 26 countries.
