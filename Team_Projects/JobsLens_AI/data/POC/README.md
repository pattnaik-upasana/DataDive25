# Data

This folder contains all datasets used in the AI job displacement analysis.

## Structure

- **raw/** - Original source data files (Excel format)
- **processed/** - Cleaned and processed datasets (generated by scripts)

## Raw Data Files

Located in `raw/`:

- **Country Data Set.xlsx** - Main World Bank I2D2 dataset
- **join_database_w_definitions.xlsx** - Data dictionary and definitions
- **most_recent_by_country_2020_2025.xlsx** - Filtered recent records (2020-2025)
- **ai_impact_analysis_results.xlsx** - Analysis results
- **country_sector_vulnerability.xlsx** - Country and sector vulnerability scores

## Data Source

World Bank International Income Distribution Database (I2D2)
- Survey years: 2020-2021
- Countries: 26 countries with recent data
- Focus: Employment, education, occupation, and sector data

## Usage

Scripts in the `scripts/` folder read from `raw/` and may generate processed outputs.

## Data Dictionary

See `join_database_w_definitions.xlsx` for variable definitions and metadata.
