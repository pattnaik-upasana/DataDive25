# Team Table4 - Challenge 4 Analysis

## Reproducing the Analysis

Due to GitHub file size limits, the data files and visualization images are not included in this repository. To reproduce the analysis:

### 1. Download Data Files
Follow the instructions in `data_instructions.md` to download:
- World Bank Worldwide Bureaucracy Indicators (WWBI)
- Female Labor Force Participation data
- Female Unemployment data

Place all CSV files in the `data/` directory.

### 2. Run Analysis Scripts

```bash
# Initial analysis (public sector distribution)
python3 analyze_data.py

# Comprehensive analysis (hypothesis testing)
python3 comprehensive_analysis.py
```

### 3. View Results
- Analysis report: `analysis.md`
- Visualizations: `results/` directory (generated by scripts)
- Top 10 countries: `results/top_10_countries.csv`

## Key Findings
- **Crowding Out Correlation**: 0.253 (weak positive)
- **Crowding In Correlation**: -0.118 (weak negative)
- **Conclusion**: Neither hypothesis is strongly supported by cross-country data
